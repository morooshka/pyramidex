# pyramidex

Replace static AI config files with a self-describing knowledge graph.
Load only what each task needs — minimal context, always current.

## Getting started

Clone the repo and open it in Claude Code:

    git clone https://github.com/morooshka/pyramidex
    cd pyramidex
    claude .

Once Claude is running, ask it to set up the project:

> Follow docs/prompts/setup.md and set up Pyramidex for me.
